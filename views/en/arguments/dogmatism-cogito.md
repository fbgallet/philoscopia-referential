# The cogito

*Argument for [Relation to certainty](../axes/certainty.md) : Dogmatism*

**Even if I doubt everything, I cannot doubt that I think.**

*[Descartes](../philosophers/descartes.md), *[Meditations on First Philosophy](../works/meditations-descartes.md)*, 1641*

## The argument

1. Let us push [doubt](../glossary/hyperbolic-doubt.md) as far as it will go: grant that an evil demon deceives me about the world, about my body, about the very truths of mathematics.
2. But to be deceived, I must still think; and if I think, I exist, at least as a thinking thing. To be wrong even about this would still require that I think.
3. The proposition "I am, I exist" is therefore true each time I frame it: doubt, far from shaking it, confirms it with every attempt to deny it.

**⇒** Therefore there exists at least one indubitable truth: doubt has a limit, and knowledge a definitive foothold.

## Deepenings

### From the cogito to the criterion of evidence

1. What makes the cogito irresistible, Descartes says, is that I conceive it **clearly and distinctly**: no obscurity in it leaves a hold for doubt.
2. Now if evidence is the mark of the true in this case, it can be so elsewhere: whatever I conceive just as clearly merits the same certainty. The cogito is not an exception, it is a model.

*[Descartes](../philosophers/descartes.md), *[Meditations on First Philosophy](../works/meditations-descartes.md)*, 1641*
