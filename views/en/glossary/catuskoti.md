# Catuṣkoṭi (tetralemma)

*Concept associated with [Nāgārjuna](../philosophers/nagarjuna.md)*

The catuṣkoṭi, or tetralemma, is a four-cornered logic that considers, for any proposition, that it may be true, false, both true and false, or neither true nor false. [Nāgārjuna](../philosophers/nagarjuna.md) uses it to refute each of these four options in turn, and so to refuse every metaphysical thesis about the ultimate nature of things. It is not one more assertion but an instrument of deconstruction in the service of [emptiness](../glossary/sunyata.md).
