# Alternate possibilities

The principle of alternate possibilities holds that one is responsible for an act only if one could have acted otherwise. It ties [freedom](../axes/freedom.md) to responsibility: it is often the route by which [free will](../glossary/free-will.md) is argued, and the thing [determinism](../glossary/determinism.md) threatens. Frankfurt contested it with famous counter-examples, where one is held responsible for an act one nonetheless could not have failed to do.
