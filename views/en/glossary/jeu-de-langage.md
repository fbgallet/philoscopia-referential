# Language-game

*Concept associated with [Wittgenstein](../philosophers/wittgenstein.md)*

The language-game (Sprachspiel) is, for the late [Wittgenstein](../philosophers/wittgenstein.md), the unit in which language means: not an abstract system, but a rule-governed activity in which words, gestures and practice interweave, such as giving orders, describing, telling a story or praying. The meaning of a word is not the object it names but its use in such a game, itself embedded in a [form of life](../glossary/forme-de-vie.md). There is an irreducible multiplicity of language-games, over which no common essence presides.
