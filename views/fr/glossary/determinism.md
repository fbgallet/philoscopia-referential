# Déterminisme

Le déterminisme est la thèse selon laquelle tout événement, y compris nos choix, résulte nécessairement de causes antérieures conformément aux lois de la nature. Si elle est vraie, nos décisions semblent fixées avant même que nous délibérions, ce qui menace [le libre arbitre](../glossary/free-will.md). Pour [Spinoza](../philosophers/spinoza.md), le sentiment de liberté vient seulement de ce que nous ignorons les causes qui nous déterminent ; le [compatibilisme](../glossary/compatibilism.md) tente, lui, de concilier cette thèse avec une forme de liberté.
