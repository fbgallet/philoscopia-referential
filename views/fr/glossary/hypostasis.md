# Hypostase

Une hypostase du grec hupostasis, ce qui se tient dessous, fondement, subsistance est un niveau de réalité qui subsiste réellement, une substance au sens fort. Dans le néoplatonisme de [Plotin](../philosophers/plotinus.md), le terme désigne les trois principes hiérarchisés qui structurent tout le réel : l'[Un](../glossary/the-one.md), l'[Intellect](../glossary/nous.md) et l'[Âme](../glossary/world-soul.md), chacun procédant du précédent par [émanation](../glossary/emanation.md). La théologie chrétienne reprendra le mot pour penser les trois personnes de la Trinité.
