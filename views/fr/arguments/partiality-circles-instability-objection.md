# Ni l'un ni l'autre

*Objection à [Partialité](../axes/partiality.md) : Cercles concentriques*

**À vouloir les deux, vous risquez de ne tenir ni l'égalité ni le lien.**

## L'argument

1. La position des cercles paraît sage, mais elle peut n'être qu'instable : ou bien l'égalité commande vraiment, et votre partialité n'est qu'une faiblesse tolérée.
2. Ou bien c'est le lien qui commande, et le « fondement impartial » n'est qu'un habillage commode de votre préférence pour les vôtres.
3. Entre les deux, on s'appuie tantôt sur l'un, tantôt sur l'autre, selon ce qui arrange : c'est peut-être ne s'appuyer sur rien.

**⇒** La voie moyenne pourrait donc n'être qu'un nom poli pour une hésitation entre deux thèses qui, elles, sont claires.

## Issues face à l'objection

- **Au fond, c'est l'égalité qui commande ; ma partialité n'est qu'une faiblesse tolérée, pas un principe.** : Vous tranchez par le haut : l'égalité est le vrai fondement, le reste un pis-aller.
- **Au fond, c'est le lien qui commande ; l'exigence impartiale n'est qu'un habillage de ma préférence pour les miens.** : Vous tranchez par le bas : le lien est le vrai fondement, l'impartialité un décor.
- **Je réfute : tenir l'égalité au fondement et la gradation dans l'acte n'est pas se contredire, c'est décrire la vie morale telle qu'elle se vit.** : Vous tenez les cercles pour une thèse, non une hésitation : un ordre, et non un flottement.
